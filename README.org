This repository contains all of my dotfiles configuration.

* Dotfiles Manager

I'm using [[https://www.gnu.org/software/stow/][GNU Stow]] a free, lightweight
dotfiles manager written in Perl to manage my dotfiles.

What differentiates it from other dotfiles managers is that it does
not require various Python, Ruby, or Perl dependencies like most dotfiles
managers.

With that, it is easy to share files among multiple users or computers with a
few command lines.

* List of configurations:

- alacritty ➔ GPU-accelerated terminal emulator
- bat ➔ A cat(1) clone with wings
- daemon-emacs ➔ Emacs daemon configuration

* Getting Started

No matter what your Linux distribution, `stow` can easily be installed according
to your package manager:

*Arch Linux*
  #+BEGIN_SRC bash
  sudo pacman -S stow
  #+END_SRC

*OpenSUSE*
  #+BEGIN_SRC bash
  sudo zypper in stow
  #+END_SRC

*Fedora*
  #+BEGIN_SRC bash
  sudo dnf install stow
  #+END_SRC

*Debian*
  #+BEGIN_SRC bash
  sudo apt-get install stow
  #+END_SRC

Once the installation is complete, make a clone of the repository:

  #+BEGIN_SRC bash
  git clone https://github.com/raksodiano/.dotfiles.git
  #+END_SRC

You can now install any configurations you wish to copy using GNU Stow:

  #+BEGIN_SRC bash
  # Make sure you are in the right directory
  cd .dotfiles

  # Example to install the htop config
  stow htop

  # Uninstall the htop config
  stow -D htop
  #+END_SRC

* Installation Script

** Usage

#+begin_src bash
# Default installation (uses paru as AUR helper)
./install.sh

# Use paru explicitly
./install.sh --paru

# Use yay as AUR helper
./install.sh --yay

# Alternative syntax
./install.sh --aur-helper paru
./install.sh --aur-helper yay
#+end_src

** Script Structure

*** =install.sh=

Main installation script that:

- Verifies the system is Arch Linux
- Updates the system with =pacman -Syu=
- Installs Git if not present
- Installs the selected AUR helper (paru or yay)
- Installs all packages from =packages.conf=
- Enables system and user services
- Configures all dotfiles using GNU Stow

*** =scripts/packages.conf=

Configuration file containing all packages organized in arrays:

| Array           | Description                              |
|-----------------+------------------------------------------|
| =SYSTEM_UTILS=  | System monitoring, file utilities, etc.  |
| =DEV_TOOLS=     | Editors, compilers, languages, LaTeX     |
| =VIRTUALIZATION=| Docker, QEMU, libvirt, virt-manager      |
| =MAINTENANCE=   | System maintenance tools                 |
| =DESKTOP=       | Window manager, terminals, browsers      |
| =OFFICE=        | Office suite                             |
| =MEDIA=         | Media players and tools                  |
| =CODECS=        | Audio/video codecs                       |
| =FONTS=         | Nerd Fonts and other fonts               |
| =SERVICES=      | System services (systemd)                |
| =SERVICES_USER= | User services (systemd --user)           |

*** =scripts/dotfiles/default.sh=

Functions to configure each dotfile using GNU Stow:

| Function              | Dotfile               |
|-----------------------+-----------------------|
| =install_alacritty=   | Alacritty terminal    |
| =install_kitty=       | Kitty terminal        |
| =install_zsh=         | Zsh + Oh My Zsh       |
| =install_starship=    | Starship prompt       |
| =install_tmux=        | Tmux multiplexer      |
| =install_doom=        | Doom Emacs            |
| =install_htop=        | htop config           |
| =install_bat=         | bat config            |
| =install_fastfetch=   | Fastfetch config      |
| =install_paru=        | Paru AUR helper       |
| =install_rmpc=        | rmpc MPD client       |
| =install_niri=        | Niri compositor       |
| =install_noctalia=    | Noctalia Shell        |
| =install_all_dotfiles=| Install all at once   |
